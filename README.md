#Online Crime Reporting System
Online Crime Reporting System Project is a software application that allows citizens to report crimes to law enforcement agencies through a website or mobile application. The system typically includes features such as crime reporting forms, location tracking, and notifications.

Online Crime Reporting System Software made with HTML, PHP, Java Script, CSS, MySql. This Software is a mini version which has the following features -

Login for Both Admin and User.
